# Structurizr Static Site

This folder contains the generated static website for the GPSD C4 architecture documentation.

The site was generated from the Structurizr workspace using Docker. Docker was used to run the Structurizr CLI in a clean and reproducible environment, without requiring a local installation of Structurizr on the system.
